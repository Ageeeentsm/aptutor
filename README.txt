AptTutor — Aptitude Test Prep Platform
=======================================

HOW TO RUN:
-----------
Windows:  Double-click START.bat
Mac/Linux: Run ./start.sh  (or: python3 app.py)

Then open: http://localhost:5000

REQUIREMENTS:
-------------
- Python 3.8+
- pip install flask   (the script does this for you)

WHAT'S INSIDE:
--------------
/ (localhost:5000)          → Landing page
/app                        → Dashboard & all features
/test                       → Take a timed or study test
/study                      → Untimed practice with instant feedback
/insights                   → Full question library with explanations
/leaderboard                → Your session rankings

NO ACCOUNT NEEDED — just open and start practising.

MODULES:
--------
• GMAT-Style    (15 questions: quantitative, critical reasoning, data sufficiency)
• SHL-Style     (15 questions: numerical, verbal, inductive)
• Watson-Glaser (13 questions: inference, assumption, deduction, evaluation)
